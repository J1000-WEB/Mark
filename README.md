# 오프라인 매출 리뷰 대시보드 Mark2.8

## 반영사항
- 금주전주 시트명 반영
- 월간 페이지 호조/부진 매장 TOP3 추가
- 월간 TOP 상품 / 점포별 월간 TOP 상품 준비중 카드 추가
- 온오프재고현황 매핑 수정: E=스타일, F=상품명, Q=온라인, R=오프라인, S=합계
- 금주전주 매핑: E=점포명, I=점포재고, W=금주합계, X=금주판매금액, AA=전주합계, AB=전주판매금액
- RT 이동 제안 카드에 보내는/받는 점포 재고수량 및 재고주수 표시
- 물류 추가 할당 제안 재계산
