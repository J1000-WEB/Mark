# Mark1 오프라인 매출 리뷰 대시보드

신규 파일 `📊 매출 통합 대시보드 2026 (NEW).xlsx`에서 추출한 정적 JSON 기반 Mark1 프로젝트입니다.

## 구성
- `/weekly` 주간 페이지
- `/daily` 일간 페이지
- `/monthly` 월간 페이지

## 특징
- V 시리즈와 같은 목표 화면
- Mark 시리즈 전용 데이터 매핑
- 외부 차트 라이브러리 없이 CSS 기반 차트
- Vercel 빌드 안정성을 위해 의존성 최소화

## 사용법
GitHub 저장소 루트에 압축 해제한 파일 전체를 업로드한 뒤 Vercel에 연결하세요.
